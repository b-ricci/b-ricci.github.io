Basic Web Design Review Cards

1) What is the correct way to set text as bold with CSS?
font-weight:bold;
2) What do you use to add more than one inline style to an element?
A semicolon
3) What is another name for a static element?
Non-positioned element
4) What is your browser's viewport?
The area in your browser where web page content is displayed
5) An element that contains another nested element is referred to as what?
A parent element
6) What is the number system used to generate digital colors?
Hexadecimal or hex code
7) CSS includes three different properties that help air out and organize webpage designs; what are they?
Padding, Margin, and Border
8) What RGB code would you use to produce cyan?
rgb(0,255,255)
